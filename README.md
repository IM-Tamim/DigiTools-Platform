# 🛠️ DigiTools — Premium Digital Tools Marketplace

A modern **digital product marketplace** built with React where users can browse, add to cart, and purchase premium tools like **AI writing assistants, design templates, stock assets, and more**. DigiTools offers a sleek, user-friendly interface with real-time feedback for a smooth shopping experience.

---

## 🌐 Live Site

[🔗 View Live](https://snazzy-florentine-44a735.netlify.app)

---

## 🧰 Technologies Used

- **⚛️ React.js** — Component-based UI for a responsive and interactive frontend  
- **🎨 Tailwind CSS** — Utility-first styling for fast, clean design  
- **🧩 DaisyUI** — Prebuilt Tailwind components for rapid development  
- **💻 JavaScript (ES6+)** — Application logic and dynamic behavior  
- **🔔 React-Toastify** — Toast notifications for user actions  
- **📄 JSON** — Local product data storage for easy management  

---

## ✨ Features

- **🛒 Cart System:**  
  Add products to cart, remove individual items, view running totals, and proceed to checkout which automatically clears the cart.  

- **🔔 Toast Notifications:**  
  Real-time feedback for add to cart, remove, and checkout actions using React-Toastify for a smoother user experience.  

- **📱 Fully Responsive:**  
  Optimized layout for mobile, tablet, and desktop screens with clean design and intuitive navigation.  

- **🔍 Product Browsing:**  
  Users can easily browse products with images, descriptions, and pricing for quick decision-making.  

- **💡 Modern UI/UX:**  
  Clean interface with Tailwind and DaisyUI components to provide an engaging shopping experience.  

- **⚡ Fast Performance:**  
  Lightweight frontend ensures fast load times and minimal lag, even with multiple product interactions.
